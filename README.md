# Color Changer Webpage

A simple, lightweight JavaScript project that allows users to change the background color of the webpage with the click of a button.

## Project Structure

The project consists of just two files:
* `main.html` - The structure of the webpage, containing the 5 color-changing buttons.
* `index.js` - The JavaScript logic that handles the click events and updates the background color.

## How to Run

1. Clone or download this repository.
2. Open `main.html` directly in any modern web browser.
3. Click any of the 5 buttons to change the background color!
